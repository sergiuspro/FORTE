Focuser commands

All focuser-related commands begin with:

o.scope.imager.foc.

Focuser commands related to its movement

1) Move the focuser by the specified value in mm relative to its current position (positive value – move outwards, negative value – move inwards), e.g.:

o.scope.imager.foc.move_by(-0.05)

2) Move the focuser to the specified position in mm relative to the home position, e.g.:

o.scope.imager.foc.move_to(0.1)

3) Move the focuser to the home position at 0 mm:

o.scope.imager.foc.reference_travel()

Focuser commands related to its state

1) Get the current focuser position in mm relative to the home position (output: positive value – outward position, negative value – inward position):

o.scope.imager.foc.get_pos()

2) Get the last focuser position, the last movement direction and the last CCD filter used (output: a 3-tuple of values; direction value 'True' – outward movement, direction value 'False' – inward movement):

o.scope.imager.foc._get_state()

Other focuser commands

1) Get the current state of the focuser (output: online|standby|suspend|offline):

o.scope.imager.foc.state

2) Get the current backlash value in mm:

o.scope.imager.foc.backlash

3) Get the current setting for the reference travel in the case of focuser initialization or focuser failure (output: True|False):

o.scope.imager.foc.ref_travel

4) Get the focuser model name:

o.scope.imager.foc.model

5) Get the focuser serial number:

o.scope.imager.foc.serial_no

6) Get the focuser termal sensor temperature in Kelvins:

o.scope.imager.foc.get_temperature()

7) Start auto-focusing:

o.scope.imager.start_task('auto_focus').join()

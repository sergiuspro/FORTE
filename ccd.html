CCD commands

All CCD-related commands begin with:

o.scope.imager.ccd.

CCD commands related to the thermoelectric cooler (TEC)

1) Check if the TEC is enabled (output: True|None):

o.scope.imager.ccd.tec_is_enabled()

2) Get the current CCD temperature in Kelvin:

o.scope.imager.ccd.get_temperature()

3) Get the current ambient temperature in Kelvin:

o.scope.imager.ccd.get_ambient_temperature()

4) Get the current TEC power in percent:

o.scope.imager.ccd.get_tec_power()

5) Regulate the CCD temperature to the given setpoint in Kelvin:

o.scope.imager.ccd.start_task('regulate_temperature',setpoint=248.15).join()

CCD commands related to the image acquisition

o.scope.imager.ccd.start_task('observe',target='',texp=int,nexp=int,exptype='',shutter='').join() #exptype OR type ????

Other CCD commands

1) Check the current state of the CCD (output: online|standby|suspend|offline):

o.scope.imager.ccd.state

2) Change the current state of the CCD:

o.scope.imager.ccd.start_task('set_state','state').join()

Here state can have one of the following values:

state='online'|'standby'|'suspend'|'offline'

3) Check if the given capability is supported by the CCD (output: True|False):

o.scope.imager.ccd.supports(cap)

Here

cap

can have one of the following values:

cap='shutter'|'tec_stabilization'|'tec_set_power'|'tec_get_power'|'tec'|'ambient_temperature'|'chip_temperature'|'trigger_input'|'trigger_output'

Installation under Ubuntu

Notice 1: FORTE (Facility for Operating Robotic TElescopes) is distributed under a proprietary license for ISON members only.

Notice 2: This manual describes FORTE version 0.21.4.20211024 which runs on Python 3.

Notice 3: Currently, this manual describes FORTE command line usage only and doesn't describe FORTE GUI and installation of its dependencies.

Installation of package dependencies: APEX

Install APEX as described in this documentation.

Installation of system dependencies: libusb

Install the package:

sudo apt install libusb-1.0-0-dev

Installation of Python dependencies: typing

Install the package:

sudo pip3 install -U typing

Installation of FORTE server

Go to the folder where the tarball forte-0.21.4.20211024.tar.gz is stored and unpack it:

tar xvzf forte-0.21.4.20211024.tar.gz

A new folder forte-0.21.4.20211024 will be created with subfolders and files unpacked into it. Change to that folder:

cd forte-0.21.4.20211024

Build it:

python3 setup.py build

Install it (notice the sudo command!):

sudo python3 setup.py install

Installation of FORTE client

Notice: Installation of FORTE client must be done only, if you plan to use FORTE GUI; for FORTE command line usage installation of FORTE server is sufficient.

Go to the folder where the tarball forte_client-0.21.4.20211024.tar.gz is stored and unpack it:

tar xvzf forte_client-0.21.4.20211024.tar.gz

A new folder forte_client-0.21.4.20211024.tar.gz will be created with subfolders and files unpacked into it. Change to that folder:

cd forte_client-0.21.4.20211024

Build it:

python3 client_setup.py build

Install it (notice the sudo command!):

sudo python3 client_setup.py install

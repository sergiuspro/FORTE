Installation under Ubuntu

Notice 1: FORTE (Facility for Operating Robotic TElescopes) is distributed under a proprietary license for ISON members only.

Notice 2: If you plan to use a FLI CCD camera along with the INDI Library driver, your computer must have Ubuntu version 15.10 or higher (check requirements at indilib.org website).

Notice 3: Currently, this manual describes FORTE command line usage only and doesn't describe FORTE GUI and its installation dependencies.

Installation of dependencies: APEX

Install APEX as described in this documentation.

Installation of dependencies: libusb

Install the libusb package:

sudo apt install libusb-1.0.0-dev

Installation of FORTE server

Go to the folder where the tarball forte-0.16.4.tar.gz is stored and unpack it:

tar xvzf forte-0.16.4.tar.gz

A new folder forte-0.16.4 will be created with subfolders and files unpacked into it. Change to that folder:

cd forte-0.16.4

Build it:

python setup.py build

Install it (notice the sudo command!):

sudo python setup.py install

Installation of FORTE client

Notice: Installation of FORTE client must be done only, if you plan to use FORTE GUI; for FORTE command line usage installation of FORTE server is sufficient.

Go to the folder where the tarball forte_client-0.16.4.tar.gz is stored and unpack it:

tar xvzf forte_client-0.16.4.tar.gz

A new folder forte_client-0.16.4.tar.gz will be created with subfolders and files unpacked into it. Change to that folder:

cd forte_client-0.16.4.tar.gz

Build it:

python client_setup.py build

Install it (notice the sudo command!):

sudo python client_setup.py install
